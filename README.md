# Chat-App

A Real-Time Chat APP Using WebSocket And [Socket.io](https://socket.io/ "socket.io library")

It allow users to easily interact with each other.

### Technologies -

- Node Js
- Express
- Web Sockets and Socket.io
- HTML
- Javascript 
- Material CSS

### Installation and Setup -

```
npm install
npm start
```

`port - 3000`
